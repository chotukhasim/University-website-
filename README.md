Overview
This is a simple multi-page website built using HTML and CSS.
It includes different sections like Home, About, Blog, and Courses to demonstrate basic web development structure and design.

Features
🏠 Home Page (index.html)
📖 About Page (about.html)
📝 Blog Page (Blog.html)
🎓 Course Page (course.html)
🎨 Custom Styling using style.css
🖼️ Image assets stored in images/ folder

Project Structure
├── images/
├── index.html
├── about.html
├── Blog.html
├── course.html
├── style.css

Technologies Used
HTML5
CSS3

Purpose
This project is created to:
Practice frontend development
Understand multi-page website structure
Improve HTML & CSS skills
